3. Добавьте две переменные:


#### Переменная 2:
```

```

### Шаг 2: Redeploy проекта

После добавления переменных:
1. Перейдите в **Deployments**
2. Нажмите на последний деплой
3. Нажмите **⋯** (три точки) → **Redeploy**

### Когда клиент заполняет форму:

1. **Данные отправляются** на `/api/submit-form`
2. **API форматирует** красивое сообщение
3. **Telegram Bot** отправляет вам уведомление

## 🔧 Локальная разработка

Для тестирования на локальном сервере:

1. Создайте файл `.env.local` в корне проекта
2. Скопируйте содержимое из `.env.example`
3. Вставьте ваши реальные токены
4. Запустите `npm run dev`

**⚠️ ВАЖНО:** Файл `.env.local` уже добавлен в `.gitignore` и не попадет в Git!

## 🐛 Troubleshooting

### Сообщения не приходят?

1. Проверьте переменные окружения на Vercel
2. Убедитесь, что вы написали `/start` боту
3. Проверьте логи в Vercel: **Deployments** → **Functions**

### Ошибка "Telegram credentials not configured"?

- Переменные окружения не установлены
- Сделайте Redeploy после добавления переменных

### Бот не отвечает?

- Проверьте, что токен правильный
- Убедитесь, что бот не заблокирован
