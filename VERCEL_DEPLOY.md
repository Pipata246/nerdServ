## ✅ Что нужно сделать

### Шаг 1: Закоммитьте изменения в Git

```bash
git add .
git commit -m "Add Telegram notifications for contact forms"
git push
```

**⚠️ ВАЖНО:** Файл `.env.local` НЕ попадет в Git (он в `.gitignore`)!

1. Зайдите на [vercel.com](https://vercel.com)
2. Откройте ваш проект
3. Перейдите в **Settings** (⚙️)

### Шаг 4: Redeploy проекта

**Вариант А: Автоматический (рекомендуется)**

Просто сделайте новый коммит:
```bash
git commit --allow-empty -m "Trigger redeploy"
git push
```

**Вариант Б: Ручной**

1. Перейдите в **Deployments**
2. Найдите последний деплой
3. Нажмите **⋯** (три точки справа)
4. Выберите **Redeploy**
5. Подтвердите **Redeploy**

---

### Шаг 5: Проверьте работу

1. Дождитесь завершения деплоя (1-2 минуты)
2. Откройте ваш сайт
3. Заполните форму обратной связи

## 🐛 Если что-то не работает

### Сообщения не приходят?

**Проверьте:**
1. ✅ Переменные окружения добавлены на Vercel
2. ✅ Вы написали `/start` боту в Telegram
3. ✅ Сделали Redeploy после добавления переменных

**Как проверить логи:**
1. Откройте проект на Vercel
2. Перейдите в **Deployments**
3. Кликните на последний деплой
4. Перейдите в **Functions**
5. Найдите `/api/submit-form`
6. Посмотрите логи ошибок

### Ошибка "Telegram credentials not configured"

- Переменные окружения не установлены
- Сделайте Redeploy

### Ошибка 400 "Bad Request"

- Проверьте, что Bot Token правильный
- Убедитесь, что Chat ID — это число без пробелов

### Бот не отвечает

- Проверьте, что бот не заблокирован
- Напишите боту `/start`
- Проверьте токен через @BotFather
